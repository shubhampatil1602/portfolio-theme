This was a project I started back in november 2020 when I first learned HTML, CSS and Javascript.

Designed this Netflix Static Clone page to learn more about CSS.

Responsive design in CSS is a crucial aspect of creating websites that adapt to different screen sizes and devices. Through media queries and flexible layout techniques, I've learned to adjust elements, fonts, and images to ensure optimal user experience across various platforms.

I've learned that Flexbox in CSS is a powerful layout model, allowing me to easily create responsive and flexible designs for web pages. It enables me to efficiently align and distribute elements within their containers, simplifying the process of building complex and dynamic layouts.
